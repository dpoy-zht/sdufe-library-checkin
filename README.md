# 🐉 奶龙图书馆远程签到系统

> **让奶龙替你签到，你只管摸鱼** 🐉✨

## 这是个啥？

山东财经大学（SDUFE）图书馆签到系统，**纯前端**，打开即用，无需登录、无需密码、无需后端。

核心原理：获取图书馆服务器时间戳 → 拼接签到URL → 手动跳转完成签到。就这么简单，简单到奶龙都会。

## 🎯 在线体验

**https://dpoy-zht.github.io/sdufe-library-checkin/**

> ⚠️ 请连接 SDUFE 校园网（`SDUFE WiFi`）后使用，否则时间戳获取会失败，奶龙也没办法。

## 🛠️ 技术架构

```
index.html  ← 一把梭，纯 HTML + CSS + JS，零依赖
├── 马卡龙色系 CSS 变量（想换皮肤改几行就行）
├── 磨砂玻璃卡片（backdrop-filter，Safari 可能会骂我）
├── 漂浮云朵动画（CSS only，3朵）
├── 闪烁星星（CSS only，5颗）
├── 奶龙摇摆动画（导航栏摇头晃脑）
├── localStorage 签到状态持久化（奶龙比你记性好）
├── Web Speech API 语音播报（奶龙会说话了！！！）
└── fetch 调用图书馆 API（被拦截了不怪我）
```

**技术栈复杂度：** 纯原生，连 jQuery 都没有，vanilla 到不能再 vanilla。

## 📡 API 原理

```
1. 请求时间戳：GET https://libst.sdufe.edu.cn/api.php/v3qrtime
   返回：{"status":1,"msg":"服务器时间","data":51526269564}

2. 拼接签到URL：http://www.skalibrary.net/wx/scancheck?school=sdufe&type={1|2|3}&t={时间戳}
   type=1 签到入馆
   type=2 临时离馆
   type=3 签离出馆

3. 用户手动点击跳转（自动跳转会丢微信Cookie，别问我怎么知道的）
```

## 🐉 设计理念

本系统遵循以下核心原则：

1. **奶龙优先** — 页面上奶龙面积 > 文字面积
2. **圆角至上** — 16px起步，40px封顶，锐角是异端
3. **马卡龙色系** — 看一眼心情就好，适合图书馆emo时刻
4. **磨砂玻璃** — blur越多越高级（苹果：你说得对）
5. **不能遮挡** — 奶龙100%不透明，谁敢加遮罩跟谁急
6. **hover必有奶龙** — 悬停弹出迷你奶龙，这是底线
7. **奶龙会说话** — 签到成功它比你先激动（Web Speech API 驱动）
8. **奶龙有情绪** — 签到状态不同，奶龙表情不同（比你的朋友圈情绪稳定多了）

## 🎙️ 奶龙语音系统 v1.0

点击操作按钮后，奶龙会开口说话：

| 操作 | 奶龙语音 | 语气 |
|------|---------|------|
| 签到入馆 | "签到成功啦~奶龙陪你学习！" | 激动、自豪，仿佛签到的不是你 |
| 临时离馆 | "临时离馆~记得早点回来哦！" | 关切但略带不安，怕你不回来了 |
| 签离出馆 | "签离出馆~奶龙明天见！" | 恋恋不舍，比毕业季还伤感 |

> 技术实现：`window.speechSynthesis`，零音频文件。音色取决于你的设备，奶龙在安卓上和iOS上声音不一样，这属于"跨平台适配"。

## 😊 奶龙情绪系统 v1.0

奶龙会根据你的签到状态**实时变脸**：

| 状态 | 奶龙表情 | 状态文字 | 潜台词 |
|------|---------|---------|--------|
| **未签到** | 催促脸🥺 | "未签到" | "你今天第几次了？" |
| **在馆学习** | 开心脸😄 | "在馆学习中" | "不愧是我看好的人" |
| **临时离馆** | 保持签到时的表情 | 不变 | "快回来，我一个人害怕" |

状态通过 `localStorage` 持久化，刷新页面奶龙依然记得你的状态。

> 这比某些社交软件的"已读不回"功能靠谱多了，至少奶龙会告诉你它在等。

## 📱 使用方法

1. 手机连接 `SDUFE WiFi`
2. 打开 https://dpoy-zht.github.io/sdufe-library-checkin/
3. 看到「已连接」就说明通了
4. 点你想点的按钮（签到/暂离/签离）
5. 听奶龙说一句鼓励的话（记得开声音）
6. 看奶龙变脸（开心/催促）
7. 弹出URL后点击「跳转到图书馆」
8. 在微信浏览器里完成签到
9. 收工，让奶龙替你值班
