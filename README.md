# 📚 SDUFE 图书馆远程签到

> 手机上点一下，随时随地签到/暂离/签离，再也不用百米冲刺赶回图书馆。

---

## 🎯 使用方式

1. 手机连接校园网 **SDUFE WiFi**
2. 在微信中打开 **[dpoy-zht.github.io](https://dpoy-zht.github.io/sdufe-library-checkin)**
3. 输入密码，点签到按钮
4. 手动点击跳转链接完成签到 ✅

---

## 🔍 原理

| 参数 | 含义 |
|------|------|
|  | 签到入馆 |
|  | 临时离馆 |
|  | 签离出馆 |
|  | 服务器时间戳（通过 API 实时获取） |

扫码二维码其实就是访问一个链接，在手机上打开它即可。

---

## ⚠️ 注意

- 必须连 **SDUFE WiFi**，否则 API 无法访问
- 需在山东财经大学图书馆公众号登录
- 暂离时间上限 100 分钟（中午除外）
